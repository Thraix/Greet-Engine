## Installation

### Install FreeType2
```
  sudo add-apt-repository ppa:glasen/freetype2
  sudo apt update
  sudo apt install freetype2-demos
```

### Install FreeImage
```
  sudo apt-get install libfreeimage3 libfreeimage-dev
```

### Install GLFW 
```
  sudo apt-get install libglfw3-dev
```

### Install GLEW
```
  sudo apt-get install libglew-dev
```
